Test de redimensionnement du texte

Cette page vous permet de tester si le site peut être redimensionné jusqu'à 200% sans perte de contenu ou de fonctionnalité.

Instructions :

  1. Utilisez les boutons dans le panneau de test (en haut à droite) pour augmenter la taille du texte.
  2. Vérifiez si tout le contenu reste visible et utilisable à chaque niveau de zoom.
  3. Portez une attention particulière aux éléments suivants :
    • Menus déroulants
    • Formulaires
    • Boutons
    • Texte dans des conteneurs à taille fixe
  4. Le panneau de test mettra en évidence les problèmes potentiels en rouge.
